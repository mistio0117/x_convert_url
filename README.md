# x_convert_url(個人用)

## 背景

twitterが X となったことに伴い、discord でTwitterのURLを添付した際に埋め込み表示がされなくなった。

## 概要

受け取ったメッセージに "https://x.com" が含めれている場合に、urlを一部置換して埋め込み表示できるようにする。
